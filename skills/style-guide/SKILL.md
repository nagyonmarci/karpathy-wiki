# Style Guide

- Every wiki file starts with YAML front-matter: `title`, `summary`, `concepts`, `sources`, `backlinks`.
- Use sentence case for headings.
- Prefer short paragraphs and bulleted lists.
- Cross-link concepts using `[[concept-name]]`.
- Code blocks must declare language.
- Cite sources at the bottom under `## Sources`.
- Keep summaries under 3 sentences.
